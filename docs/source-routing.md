# Source Routing

Use this before reading deeper source documents.

This is a support doc for the builder workflow. It is not an episode, scene, guest, source-reference or handoff output.

## Starting Point

Use Luke's local `_agent_navigation` layer if it is available in the working environment.

For public-facing outputs, record the topic lane and source title, not raw local file paths.

## Useful Lanes For This Repo

- Film / music / storytelling / media
- Minjerribah / Straddie local systems
- Aura / digital twins / personal AI
- Software / agents / simulation architecture
- Regenerative civilisation / protopian systems
- Governance / politics / public policy

## Token-Saving Rule

Do not scan the whole source archive.

Pick the nearest lane, read the top mind map if one exists, then open only one mature anchor document if the episode genuinely needs it.

## Red Dog Use

Red Dog can draw from deeper datasets, but the episode note should translate the source into plain yarn language.

If the source context is too large, use the Handoff form in `builders/index.html` for a later source pass.
