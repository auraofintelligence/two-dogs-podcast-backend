# Guests

Use this folder for generated real guest planning notes.

Create these files through [../builders/index.html](../builders/index.html), using the Guest form.

## Important

Guests choose their own spirit animal character and nickname.

This repo can ask helpful questions and record what they choose. It should not assign an animal, nickname or symbolic identity for them.

## Blue Dog Boundary

If a guest note expects Blue Dog involvement, write only:

`[Blue Dog question or response space - Angel to direct]`
