# Scenes

Use this folder for generated animated CGI/AI scene ideas.

Scenes can come from:

- episode outlines
- theme song images
- guest stories
- sponsor reads
- Strange But True future-world prompts
- quick jokes that deserve a visual beat
- recurring show bits from [../docs/recurring-scenes.md](../docs/recurring-scenes.md)

Create these files through [../builders/index.html](../builders/index.html), using the Scene form.
