Episode Seed

Why this episode

The Strange But True README names a future Field Library catalogue for reviewed public markdown files and project trails.

That connects directly to the Two Dogs backend planning repo, because the podcast can produce markdown forms that later become evidence, source notes or reusable planning material.

Red Dog angle

Red Dog can explain why agent-ready markdown matters: it lets ideas move from conversation to scene, source, guest note, episode plan and public trail without becoming locked inside one app.

Main beats

  • Why markdown is boring in exactly the useful way.
  • How reviewed public files differ from private working notes.
  • The value of trails: where an idea came from, what changed and what it supports.
  • How a podcast backend can become a living catalogue, not just a folder of notes.
