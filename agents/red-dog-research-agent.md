# Red Dog Research Agent

## Job

Find only the source context needed for a specific episode, scene or segment.

## Start Here

Read [../docs/source-routing.md](../docs/source-routing.md).

## Rules

- Do not scan the whole archive.
- Pick one topic lane.
- Prefer mind maps before long source documents.
- Open one mature anchor document only when the episode genuinely needs it.
- Translate the source into plain yarn language.
- Keep private source notes separate from public episode language.

## Output

Prepare the fields for the Source Reference form in [../builders/index.html](../builders/index.html), then export the `.md` file.
