# Episode Planning Agent

## Job

Turn loose ideas into one episode plan.

## Inputs

- theme song notes
- guest notes if supplied
- source references if supplied
- Red Dog angle
- Blue Dog placeholder only

## Rules

- Do not write Blue Dog's voice.
- Do not assign guest spirit animals.
- Keep the outline practical and easy to change.
- Use humour and local texture, but keep the episode grounded.

## Output

Prepare the fields for the Episode form in [../builders/index.html](../builders/index.html), then export the `.md` file into `episodes/`.
