# Ads and Sponsors

Use this folder for generated sponsor-fit notes, ad read ideas and playful scene-based ads.

Create these files through [../builders/index.html](../builders/index.html), using the Ad/Sponsor form.

## Fit Test

A sponsor idea should feel like it belongs in the Two Dogs world.

Useful fits:

- local food, events, artists, makers and services
- filming, audio, production and creative tools
- community helpers
- practical future tools explained in plain English
- support bundles or audience-backed experiments

Avoid ad ideas that clash with the grounded, kind, playful tone.
