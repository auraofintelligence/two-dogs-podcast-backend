# Handoffs

Use this folder for small task packets.

Create handoff files through [../builders/index.html](../builders/index.html), using the Handoff form.

A handoff should tell the next agent:

- the exact task
- the file to work from
- the source files allowed
- what not to touch
- what output to create

Keep it short. If a handoff needs the whole archive, it is probably too big.
